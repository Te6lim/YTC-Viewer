# YTC-Viewer
A project that demonstrates consuming data via http requests, with the use of important libraries such as Retrofit2, Paging3 and Room
## Libraries Used
- Retrofit2
- Moshi
- OkHttpInterceptor
- Glide
- Room
- Paging3
- RecyclerView
- Navigation
